 #餐廳清單
此專案提供使用者瀏覽餐廳相關資訊。

 ##功能列表

* 瀏覽餐廳列表，得知餐廳類別與評分
* 點擊`餐廳圖示`可瀏覽餐廳詳細資訊
* 點擊餐廳地址右側`箭頭圖示`，可連結至Google地圖查看餐廳位置
* 可使用關鍵字搜尋餐廳名稱

 ###安裝
1.開啟Terminal終端機，將專案clone到本機伺服器
  git clone https://github.com/bluelsa/restaurant-list
2.初始
* 切至專案資料夾
  cd restaurant-list
* 安裝套件
  npm install
3.開啟程式
  npm run start

 ##使用工具
